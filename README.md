##Let's Code!
1.Make sure you have installed the dependencies
  * `git`
  * `node`
  * `npm install -g karma-cli`

2.Clone this respository
```sh
  $ git clone https://github.com/Kanokorn/CaptchaJS.git
  $ cd CaptchaJS
```

3. Install
```sh
  $ npm install 
```
4. Run Test

```sh
  $ karma start
```
